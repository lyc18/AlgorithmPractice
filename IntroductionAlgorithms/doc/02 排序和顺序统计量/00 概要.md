# 各排序算法

| 算法     | 最坏情况运行时间 | 平均情况/期望运行时间 |
| -------- | ---------------- | --------------------- |
| 插入排序 | n^2              | n^2                   |
| 归并排序 | nlgn             | nlgn                  |
| 堆排序   | nlgn             |                       |
| 快速排序 | n^2              | nlgn(期望)            |
| 计数排序 | k+n              | k+n                   |
| 基数排序 | d(k+n)           | d(k+n)                |
| 桶排序   | n^2              | n(平均情况)           |
